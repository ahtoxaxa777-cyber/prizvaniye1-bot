# ШПАРГАЛКА - БЫСТРЫЕ КОМАНДЫ

## ЗАГРУЗИТЬ НА GITHUB (первый раз)

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/твой_username/prizvaniye-bot.git
git push -u origin main
```

## ОБНОВИТЬ КОД (после изменений)

```bash
git add .
git commit -m "Обновил тексты/ссылки/настройки"
git push
```

Railway автоматически перезапустит бота.

---

## ТЕСТИРОВАТЬ ЛОКАЛЬНО (на своём компе)

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить бота
python bot.py
```

Бот запустится на твоём компе. Ctrl+C чтобы остановить.

---

## БЫСТРО ПОМЕНЯТЬ ТЕКСТЫ

1. Открой `texts.py`
2. Найди нужный текст (Ctrl+F)
3. Измени
4. Сохрани (Ctrl+S)
5. Загрузи на GitHub:
   ```bash
   git add texts.py
   git commit -m "Обновил тексты"
   git push
   ```

---

## БЫСТРО ПОМЕНЯТЬ ССЫЛКИ

1. Открой `config.py`
2. Найди нужную ссылку
3. Вставь свою
4. Сохрани
5. Загрузи на GitHub (команды выше)

---

## ПОСМОТРЕТЬ ЛОГИ НА RAILWAY

1. Зайди на railway.app
2. Открой свой проект
3. Вкладка "Deployments"
4. "View Logs"

Там видно всё что происходит с ботом.

---

## ПОСМОТРЕТЬ БАЗУ ДАННЫХ

```bash
# Подключиться к Railway
railway login
railway link

# Открыть shell
railway shell

# Посмотреть БД
sqlite3 bot.db

# Команды:
SELECT * FROM users;                        # Все пользователи
SELECT * FROM tags WHERE tag='paid';        # Кто оплатил
SELECT COUNT(*) FROM users;                 # Сколько всего
SELECT COUNT(*) FROM tags WHERE tag='paid'; # Сколько купили

# Выйти
.quit
```

---

## ОСТАНОВИТЬ/ЗАПУСТИТЬ БОТА

### В Railway:
1. Открой проект
2. Вкладка "Settings"
3. Кнопка "Pause Service" / "Resume Service"

### Локально:
- Запустить: `python bot.py`
- Остановить: `Ctrl+C`

---

## ЧАСТЫЕ ДЕЙСТВИЯ

### Изменить цену курса
```python
# config.py
COURSE_PRICE = 9900  # меняй здесь
```

### Изменить таймеры триггеров (для теста)
```python
# config.py
TRIGGER_1_DELAY = 60     # 1 минута
TRIGGER_2_DELAY = 120    # 2 минуты
# и т.д.
```

### Добавить новый модуль
```python
# config.py
NOTION_MODULE2 = "https://notion.so/твоя_ссылка"
```
